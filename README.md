本篇论文全文使用`Latex`语法写成，包括公式，表格与图片以及文献参考
论文题目为`ISAC 技术在无线网络领域的架构设计与技术策略`，包含
```
1.引言/Introduction
2.ISAC的历史发展
3.ISAC的性能权衡
4.通信辅助感知：感知网络
5.ISAC相关的产业活动和研究成果
6.总结与展望/ConclusIon And RecommendAtIons
```
六部分

第 1 章绪论：介绍 ISAC 的基本概念，将其视为推动 B5G/6G 新兴应用的关键基础设施。阐述了感知与通信系统在硬件架构、信道特性及信号处理方面日益增强的相似性，以及集成后的频谱效率和硬件效率优势；
第 2 章回顾了从早期军用雷达脉冲调制到相控阵技术，再到 MIMO 技术与 OFDM 技术在感知中应用的历史演进；
第 3 章探讨了 ISAC 的理论基石。从物理层指标（检测概率、估计均方误差 MSE、克拉美罗下界 CRB）到信息论极限（I-MMSE 关系、容量-失真权衡），揭示了通信速率与感知精度之间帕累托（Pareto）最优边界的理论模型；
第 4 章分析了基于 5G NR 帧结构进行感知的可行性。详细讨论了利用同步信号块（SSB）、解调参考信号（DMRS）和定位参考信号（PRS）进行感知的机制，并指出了高精度感知面临的带宽受限、自干扰及双基地感知中的未知数据负载等挑战； 
第 5 章介绍了 ISAC 在增强定位与跟踪、区域成像（射频成像）以及车联网（V2X）中的具体应用。重点探讨了 ISAC 如何辅助车辆编队行驶，以及通过同步定位与建图（SLAM）提升态势感知能力。
